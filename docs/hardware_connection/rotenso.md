# Rotenso Hardware Connection

## Scope

Vendor-level guidance for Rotenso residential lineup expected to share Plum ecoVENT backbone.

## Current compatibility stance

- Residential lineup is considered likely compatible.
- Exact model list and per-model validation are still in progress.

## Connection checklist

- Confirm local network availability to unit controller
- Confirm Modbus TCP endpoint/port and Unit ID
- Confirm no network segmentation blocks Home Assistant host

## Home Assistant setup hints

- Use default trial values first (`port=502`, `unit=1`)
- Validate sensors and diagnostics before enabling control writes
